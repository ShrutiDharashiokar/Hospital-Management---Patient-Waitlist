# Hospital Management: Patient-Waitlist
Interactive healthcare dashboard analyzing inpatient and outpatient waiting list trends, age profiles, and specialty-level insights for operational planning.

# Description
Developed a healthcare analytics dashboard to monitor patient waiting list data across inpatient, outpatient, and day-case categories. The solution combines appended datasets, specialty mapping, and time-band analysis to track waitlist trends, patient demographics, and total volumes. Key measures such as average, median, and total waitlist support data-driven healthcare planning and resource allocation.

# Features & Visuals
Latest month vs previous year waitlist KPI cards
Time band and age profile patient breakdown
Donut chart for day case, inpatient, outpatient distribution
Line charts tracking waitlist trends by case type
Date, specialty, and case type slicers for flexible analysis
Paginated tooltip for detailed insights

# Business Problem
Healthcare teams needed clearer visibility into patient waitlists to track trends, manage demand, and improve resource planning across inpatient and outpatient services.
